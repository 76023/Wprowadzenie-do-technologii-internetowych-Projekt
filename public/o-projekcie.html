O projekcie

ISACzytac powstaje w ramach przedmiotu Wprowadzenie do technologii internetowych.

Cel aplikacji

Aplikacja pozwala prowadzić prostą listę książek, dodawać własne pozycje, komentować je i filtrować katalog według tytułu, autora oraz kategorii.

Technologie

  • Frontend: HTML, CSS i JavaScript bez dodatkowego frameworka.
  • Backend: Node.js oraz Express.
  • Baza danych: SQLite z tabelami użytkowników, książek, kategorii i komentarzy.
  • Logowanie: sesje serwerowe oraz hasła zapisane jako hash bcrypt.

Zakres projektu

Projekt został przygotowany jako kompletna, ale nadal prosta aplikacja semestralna. Kod celowo nie używa rozbudowanej architektury, żeby łatwo było wyjaśnić działanie tras Express, zapytań SQL i komunikacji frontendu z API.
